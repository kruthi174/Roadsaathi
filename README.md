# Roadsaathi
On-demand roadside assistance platform for India — two-wheeler-first, EV-native, built with PostGIS-based dispatch matching.
🔗 **Live Prototype:** https://kruthi174.github.io/Roadsaathi/roadsaathi-prototype.html
## What's in this repo
- `roadsaathi-prototype.html` — clickable UI prototype (rider + provider app flows)
- `roadsaathi-schema.sql` — PostgreSQL/PostGIS schema + dispatch-matching function
- `roadsaathi-master-plan.md` — full build roadmap and checklist
- `india-roadside-assistance-roadmap.md` — market research and business plan
